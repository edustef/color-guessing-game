How to play?

×

The main point of the game is to find out the right color that is specified in the RGB()

RGB stands for red, green, blue. In computers they are used to create all kind of colors.

The numbers are used to represent the amount of that color. They go from 0(black) to 255(white). So a color with RGB(255,0,0) is all red, no blue, no green. Which means pure red. (The colors are always in this order. RED, GREEN, BLUE)

In this game you have to guess the color that is specified by the top title. You have limited number of tries so watch out.

The system point is like this:

- Everytime you guess wrong you will lose 1 point

- If you fail to find the color you lose 5 points

- Finding the color first try gives you 5 points

- Finding the color not in a first try gives you 4 points
